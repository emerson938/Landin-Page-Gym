# Landin-Page-Gym

🏋️‍♂️ Landing Page - Academia
📌 Descrição

Este projeto consiste em uma landing page para uma academia, desenvolvida utilizando HTML e CSS.
O objetivo é apresentar os serviços, planos e informações da academia de forma moderna e responsiva.

🎯 Objetivo

Projeto desenvolvido como solução para um desafio prático de front-end, com foco em estruturação de layout e estilização.
